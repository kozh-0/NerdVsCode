# Запуск

| Команда                                              | Значение                    |
| ---------------------------------------------------- | --------------------------- |
| `git clone https://github.com/kozh-0/NerdVsCode.git` | Копирование репозитория     |
| `mvn install`                                        | Установка java зависимостей |
| `docker compose up -d`                               | Поднятие докер контейнеров  |
